Just A Rather Very Intelligent System

A sophisticated hybrid AI personal assistant with voice-activated system automation, hardware monitoring, and natural language interaction — built on a triple-layer brain architecture inspired by Iron Man.

  • Fail-safe hybrid LLM layer switching between Google Gemini (cloud) and Ollama/Llama 3.2 (local) based on connectivity
  • Custom intent parser translating voice commands into structured JSON for system-level automation — app launching, file management, auto-typing
  • Futuristic Tkinter GUI with dynamic ARC visualizer reacting in real time to system load and audio input
  • Python threading for simultaneous audio capture, TTS output, and UI updates without blocking
  • Real-time hardware telemetry (CPU, RAM, battery) via psutil with voice-activated diagnostics

Superior Messenger

University-scale cross-platform chat network

A full-stack cross-platform real-time messaging application with a Flutter frontend and Node.js/Express backend — built exclusively for Superior University students and staff.

  • Socket.io for real-time bi-directional communication enabling instant message delivery across all platforms
  • SQLite for both local (client-side) and server-side data persistence — offline support and fast queries
  • Google Sign-In for secure OAuth-based user authentication tied to university emails
  • Provider pattern for scalable, maintainable state management across the Flutter app
  • Dashboard analytics showing total users, online status, recent activity feed, and quick actions
